cs5272_project

An embedded software design project. The project was based on the embedded hardware platform ODROID-N2 which is capable of heterogeneous processing. It consists of 4 ARM A73 cores, 2 ARM A53 cores and one ARM Mali-G52 GPU. The runtime management strategy is implemented as a userspace governor. It introduces a power-performance tradeoff strategy to a neural network object detection application running on the hardware platform.

The userspace governor capable of: 1. adjusting the DVFS of CPU clusters to ensure power efficiecny 2. ensuring that the the QoS (in terms of FPS) of an machine learning objection detection application is maintained.
